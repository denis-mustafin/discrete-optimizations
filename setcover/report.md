# Решение задачи о покрытии множествами

## Структура репозитория

- `main.cpp` - код текущего решения. Читает со стандартного ввода входной файл. В первой строке выводит счет решения на тесте, во второй - количество взятых множеств, в третьей - индексы самих множеств в 0-индексации.
- `check.cpp` - код чекера. Аргументами командной строки получает пути до входного и выходного файлов. Проверяет корректность вывода и выводит его счет.
- `main` и `check` - скомпилированные (под Линукс) версии `main.cpp` и `check.cpp`
- `scored_data` - папка с оцениваемыми тестами
- `scored_data_output` - папка с ответами на оцениваемые тесты.

Пример использования чекера: `./check scored_data/sc_157_0 scored_data_output/sc_157_0.out`

## Текущее решение

Сейчас написан только несложный жадник, нужный чтобы инициализировать ответ. 

### Описание жадника

Для каждого множества поддерживаем его удельную стоимость: отношение его цены к количеству элементов в нем, не покрытых уже взятыми множествами.
Каждый раз берем множество с минимальной удельной стоимитью, добавляем его в ответ и обновляем удельные стоимости остальных множеств.

Такое решение уже набирает первую границу на всех тестах кроме `sc_330_0` и во многих приближается ко второй.
